Le projet consiste à créer un minichat avec le langage PHP et  a pour objectif de mettre en pratique les connaissances sur le langage PHP et MYSQL.
Ce minichat doit afficher les 10 premiers messages dans une page, les 10 suivants dans une autre  et ainsi de suite.

NB: Une page ne doit contenir que 1O messages et pas plus. L'accès à une page se fera simplement par la création de liens
